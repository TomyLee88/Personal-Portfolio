Portfolio Platform – Real-World Features & Deployment


GitHub

Personal portfolio

This project is not just a static portfolio — it's a fully functional web application built with real-world development practices. The goal was to create a professional platform that demonstrates secure backend integration, API usage, and automated deployment.

Tech Stack

Backend: Django (Python)
Frontend: HTML, CSS, Bootstrap
API: GitHub API (authenticated)
Deployment: Render
Security: Environment Variables, OAuth (Google)

Key Features

  • Secure Contact Form: Uses Gmail OAuth and environment variables to protect credentials.
  • GitHub API Integration: Fetches and displays projects dynamically from authenticated API requests.
  • Latest Projects Section: Homepage shows most recent GitHub repositories in real time.
  • Auto Deployment: Render listens for GitHub pushes and updates the live site automatically.

Challenges Solved

  • Protected secret keys using environment variables.
  • Handled GitHub rate limits with authorized API access (5000 requests/hour).
  • Implemented email functionality without exposing credentials.
  • Set up continuous deployment for real-world workflow.

Outcome

The portfolio showcases my skills while functioning as a real, secure, dynamic application. It allowed me to apply deployment practices, improve backend security, and integrate third-party APIs professionally.

Event Network

This is a social web platform where users can register, create posts and events, connect with friends, and interact through a full CRUD and friendship system. This project demonstrates my ability to build a functional social application with multiple features in a short timeframe.

Tech Stack

  • Backend: Python / Django
  • Frontend: JavaScript / HTML / CSS / Bootstrap
  • Database: SQLite
  • Deployment: Render

Key Features

  • User registration and authentication
  • Friendship system: send requests, accept, remove, delete friends
  • Search functionality for users, posts, and events
  • Post and Event creation with full CRUD operations
  • Basic social networking features

Challenges Solved

Registration & Authentication: Implemented a secure login system using Django, handled form validation, and managed user sessions.
Friendship System: Managed friend requests, accept/remove logic, prevented duplicates and self-requests.
Search & Filtering: Built efficient queries for users, posts, and events.
Profile & Home Page: Rendered dynamic content for posts, events, and user data with responsive layout.
Data Integrity: Ensured all CRUD operations were consistent and safe.

Role

Lead Developer: Solo Project, completed in 1–2 months
